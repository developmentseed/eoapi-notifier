# eoAPI Notifier

Message handler for eoAPI components. A middleware tool that listens to sources for messages and forwards them to output receivers.

## Installation

Install using `uv`:

```bash
uv add eoapi-notifier
```

## Testing

Install test dependencies and run tests with `uv`:

```bash
uv sync --extra test
uv run pytest
```
